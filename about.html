About Plant Care Assistant

Learn more about this educational project

Project Overview

Plant Care Assistant is a comprehensive frontend-only web application designed as an academic project. It helps users discover suitable plants for their climate and provides expert plant care advice based on their geographical location.

The project demonstrates modern web development practices using vanilla HTML, CSS, and JavaScript without any backend dependencies.

Technology Stack

Frontend
  • HTML5
  • CSS3 (Flexbox, Grid)
  • Vanilla JavaScript (ES6+)
External APIs
  • IP-API - Location detection
  • OpenWeather API - Weather data
Features
  • No backend server
  • No database
  • No npm packages
  • Works with VS Code Live Server

Key Features

  • Automatic country detection using IP-based geolocation
  • Real-time weather information for your location
  • Database of plants specific to different countries
  • Common plant problems and solutions
  • Comprehensive plant care guides
  • Fully responsive design for all devices
  • Fast loading with optimized frontend code
  • Clean, user-friendly plant-themed UI

How It Works

1

Detect Location

The app uses IP-API to automatically detect your country and location based on your IP address.

2

Fetch Weather

Real-time weather data is retrieved from OpenWeather API using your location coordinates.

3

Display Plant Data

Based on your country, the app displays suitable plants, common problems, and care solutions.

4

Provide Recommendations

Personalized care recommendations are generated based on your weather conditions.

Setup Instructions

  1. Extract the project folder to your desired location
  2. Open the project folder in VS Code
  3. Install the "Live Server" extension if not already installed
  4. Right-click on index.html and select "Open with Live Server"
  5. The application will open in your default browser
  6. For full weather functionality, get a free API key from OpenWeather and update js/weather.js
